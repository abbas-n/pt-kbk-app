# سامانه پیک و سفیران

این پروژه یک سامانه مدیریت پیک و سفیران است که با Next.js و Material-UI ساخته شده است.

## ویژگی‌ها

- رابط کاربری مدرن و رسپانسیو
- پشتیبانی از زبان فارسی و راست به چپ
- صفحات مختلف شامل:
  - صفحه لاگین
  - داشبورد
  - لیست سفیران
  - پروفایل کاربری

## پیش‌نیازها

- Node.js (نسخه 14 یا بالاتر)
- npm یا yarn

## نصب و راه‌اندازی

1. کلون کردن پروژه:
```bash
git clone [آدرس-ریپازیتوری]
cd courier-system
```

2. نصب وابستگی‌ها:
```bash
npm install
# یا
yarn install
```

3. اجرای پروژه در حالت توسعه:
```bash
npm run dev
# یا
yarn dev
```

4. باز کردن مرورگر و رفتن به آدرس:
```
http://localhost:3000
```

## ساختار پروژه

```
src/
├── app/
│   ├── components/     # کامپوننت‌های قابل استفاده مجدد
│   ├── pages/         # صفحات اصلی برنامه
│   ├── hooks/         # هوک‌های سفارشی
│   ├── contexts/      # کانتکست‌های React
│   ├── utils/         # توابع کمکی
│   ├── types/         # تعریف تایپ‌ها
│   └── styles/        # استایل‌های سفارشی
```

## تکنولوژی‌های استفاده شده

- Next.js
- React
- Material-UI
- TypeScript
- Emotion (برای استایل‌دهی)
